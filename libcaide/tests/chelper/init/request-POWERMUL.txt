        <div class="problem-statement mathjax-support" id="problem-statement">

          <h3 class=" mathjax-support"> Read problems statements in <a target="_blank" href="http://www.codechef.com/download/translated/NOV14/mandarin/POWERMUL.pdf" class=" mathjax-support">Mandarin Chinese </a> and <a target="_blank" href="http://www.codechef.com/download/translated/NOV14/russian/POWERMUL.pdf" class=" mathjax-support">Russian</a>.</h3>


You are given a function f which is defined as :<br class=" mathjax-support"><br class=" mathjax-support">
<img src="https://codechef_shared.s3.amazonaws.com/download/NOV14/CodeCogsEqn%283%29.gif" class=" mathjax-support"><br class=" mathjax-support"><br class=" mathjax-support">

Your task is to find the value of  <b class=" mathjax-support">f(N) / ( f(r)*f(N-r) )</b><br class=" mathjax-support"><br class=" mathjax-support"> . As it will be a large number output it modulo <b class=" mathjax-support">M</b>.
<h3 class=" mathjax-support"> Input </h3>

First line contains <b class=" mathjax-support">T</b> , number of test cases to follow.<br class=" mathjax-support">

Next follows <b class=" mathjax-support">T</b> test case.<br class=" mathjax-support">

First line of ever test case contain 3 space separated integers <b class=" mathjax-support">N , M</b> and <b class=" mathjax-support">Q</b>.<br class=" mathjax-support">

Next <b class=" mathjax-support">Q</b> line follows , each line contain r.<br class=" mathjax-support">

<h3 class=" mathjax-support">Output</h3>

For every test case , output <b class=" mathjax-support">Q</b> lines , each line containing the answer.<br class=" mathjax-support">

<h3 class=" mathjax-support">Constraints</h3>
<ul class=" mathjax-support">
<li class=" mathjax-support">2 ≤  <b class=" mathjax-support">N</b>  ≤ 10<sup class=" mathjax-support">5</sup></li>
<li class=" mathjax-support">1 ≤ <b class=" mathjax-support">M</b> ≤ 10<sup class=" mathjax-support">9</sup></li>
<li class=" mathjax-support">2 ≤ Sum of <b class=" mathjax-support">N</b> over all test cases ≤ 5*10<sup class=" mathjax-support">5</sup></li>
<li class=" mathjax-support">2 ≤ Sum of <b class=" mathjax-support">Q</b> over all test cases ≤ 10<sup class=" mathjax-support">5</sup></li>
<li class=" mathjax-support">1 &lt; <b class=" mathjax-support">r</b> &lt; <b class=" mathjax-support">N</b></li>
</ul>

<h3 class=" mathjax-support">Subtasks</h3>
<ul class=" mathjax-support">
<li class=" mathjax-support"> Subtask 1:  N ≤ 5 , rest of the constraints are same. Points - 9 </li>
<li class=" mathjax-support"> Subtask 2:  N ≤ 500 , M is a prime number , rest of the constraints are same. Points - 31 </li>
<li class=" mathjax-support"> Subtask 3: Refer to constraints above : Points-60 </li>
</ul>

<pre class=" mathjax-support"><h3 class=" mathjax-support">Sample Input</h3>
1
5 24 2
2
3
<h3 class=" mathjax-support">Sample Output</h3>
8
8
</pre>
<h3 class=" mathjax-support">Explanation</h3>
f[1] = 1<br class=" mathjax-support">
f[2] = 4<br class=" mathjax-support">
f[3] = 1*2<sup class=" mathjax-support">2</sup>*3<sup class=" mathjax-support">3</sup> = 108<br class=" mathjax-support">
f[4] =1*2<sup class=" mathjax-support">2</sup>*3<sup class=" mathjax-support">3</sup>*4<sup class=" mathjax-support">4</sup> = 27648<br class=" mathjax-support">
f[5] = 1*2<sup class=" mathjax-support">2</sup>*3<sup class=" mathjax-support">3</sup>*4<sup class=" mathjax-support">4</sup>*5<sup class=" mathjax-support">5</sup> =86400000<br class=" mathjax-support">
value of f[5] / (f[2]*f[3]) = 200000 and 200000%24 is 8<br class=" mathjax-support">
